## 🔧 Installation

### Setup (Recommended)
1. Download `GameStringer-1.0.9-Setup.exe`
2. Run the installer
3. Follow the installation wizard
4. Launch GameStringer from Start Menu

### Portable
1. Download `GameStringer-1.0.9-Portable.zip`
2. Extract to any folder
3. Run `GameStringer.exe`
