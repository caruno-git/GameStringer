# 🚀 Riferimento Rapido - Sistema Profili GameStringer

## ⚡ Azioni Rapide

### Primo Avvio
1. **Avvia GameStringer** → Vedrai schermata creazione profilo
2. **Inserisci nome e password** → Clicca "Crea Profilo"
3. **Configurazione automatica** → Sei pronto!

### Uso Quotidiano
1. **Avvio** → Seleziona profilo → Inserisci password → Accedi
2. **Cambio profilo** → Menu profilo → "Cambia Profilo"
3. **Logout** → Menu profilo → "Logout"

### Backup Essenziale
1. **Export** → Menu profilo → "Gestisci Profili" → "Esporta"
2. **Import** → Schermata selezione → "Importa Profilo"

## 🔒 Sicurezza Rapida

### ✅ Cosa Fare
- ✅ Usa password diverse per ogni profilo
- ✅ Fai backup regolari (export profili)
- ✅ Logout quando non usi l'app
- ✅ Mantieni GameStringer aggiornato

### ❌ Cosa NON Fare
- ❌ Non condividere password profili
- ❌ Non usare password troppo semplici
- ❌ Non eliminare profili senza backup

---

## Problemi Comuni

### "Password Errata"
- **Verifica**: Caps Lock, tastiera corretta
- **Soluzione**: Usa la Recovery Key (12 parole) per reimpostare la password

### "Profilo Non Si Carica"
- **Riavvia** l'applicazione
- **Importa** da backup se disponibile

### "Credenziali Non Salvate"
- **Verifica** di essere nel profilo corretto
- **Riprova** dopo logout/login

### "App Lenta"
- **Troppi profili**: Elimina quelli non usati
- **Cache piena**: Pulisci cache nelle impostazioni

## Migrazione Veloce

### Da Versione Precedente
1. **Aggiorna** GameStringer alla 1.3.0+
2. **Avvia** → Vedrai wizard migrazione
3. **Segui** le istruzioni automatiche
4. **Verifica** che tutto sia migrato
5. **Fai backup** del nuovo profilo

### Problemi Migrazione
- **Dati non trovati**: Controlla posizione file vecchi
- **Migrazione fallita**: Prova migrazione manuale
- **Credenziali perse**: Riconfigura manualmente
