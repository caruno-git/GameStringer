## 置信度热图

通过颜色编码的地图可视化每个翻译的质量，即时识别有问题的字符串。

### 功能

- **8项分析指标**：缺少占位符、空字符串、未翻译、标点符号、大写字母、HTML标签、长度、数字
- **颜色代码**：
  - 🟢 **优秀** (90–100%) — 正确翻译
  - 🔵 **良好** (75–89%) — 小型风格问题
  - 🟡 **可接受** (60–74%) — 轻微问题
  - 🟠 **需审查** (40–59%) — 重大错误
  - 🔴 **差** (<40%) — 严重错误
- **3种输入模式**：内置演示、粘贴文本（每行`原文|翻译`）、上传文件（JSON/CSV/TXT）
- **导出报告**：下载包含每个字符串分数和问题的JSON

### 使用方法

1. 前往**高级工具 → 置信度热图**
2. 选择模式：**演示**查看示例，**粘贴**手动输入，**文件**上传
3. 点击**"分析"**
4. 查看颜色报告：红色/橙色字符串需要优先修订
5. 使用**"导出报告"**将结果保存为JSON

## 博客管理

管理翻译项目的新闻和更新博客，在仪表板上可见。

### 功能

- **创建帖子**：标题、日期、简短描述、类别标签
- **可用标签**：Feature, UI, Fix, Security, AI, Update, News
- **置顶**：将重要帖子固定在列表顶部
- **内联编辑**：无需更改页面即可编辑任何帖子
- **删除帖子**：确认后删除
- **显示**：带有风格化日期、彩色标签徽章和描述预览的时间顺序列表

### 使用方法

1. 从主菜单前往**博客管理**
2. 点击**"新帖子"**
3. 填写日期（例如"24 Jan"）、标题（推荐使用表情符号）、描述和标签
4. 点击**"保存"**
5. 使用📌图标将帖子置顶

## Ren'Py 补丁工具

专为使用Ren'Py引擎构建的视觉小说设计的补丁工具。从`.rpy`文件中提取对话、菜单和叙述，并生成原生翻译文件。

### 功能

- **自动检测**：识别游戏的标题、版本和脚本文件
- **字符串类型**：对话、菜单、叙述
- **角色识别**：显示哪个角色说每句话
- **内联编辑器**：点击任何字符串编辑翻译
- **搜索和筛选**：按文本或角色搜索，按类型筛选
- **生成`.rpy`文件**：创建与Ren'Py兼容的`tl/<语言>/`结构
- **保存/加载JSON**：保存进度并稍后恢复
- **统计**：完成百分比，按类型计数

### 使用方法

1. 从侧边栏前往**Ren'Py 补丁工具**
2. 点击**"浏览"**并选择Ren'Py游戏文件夹
3. 点击**"提取字符串"**
4. 在编辑器中编辑翻译（点击字符串进行编辑）
5. 输入目标语言名称（例如`chinese`）并点击**"生成.rpy"**
6. 文件保存在游戏的`tl/`文件夹中

## Danganronpa 补丁工具

专为Danganronpa系列游戏设计的补丁工具。处理`.pak`档案和`.po`本地化文件。

### 功能

- **游戏检测**：自动识别DR1、DR2、V3
- **PAK档案**：提取并列出`.pak`档案中的文件
- **PO文件**：原生支持`.po`/`.pot`文件，带有已翻译/未翻译/模糊状态
- **内置AI翻译**：用配置的AI自动翻译字符串的按钮
- **PO统计**：已翻译、未翻译、模糊计数和百分比
- **DRAT集成**：高级档案操作的DRAT工具链接
- **导出补丁**：导出修改后的`.po`文件

### 使用方法

1. 从侧边栏前往**Danganronpa 补丁工具**
2. 选择Danganronpa游戏文件夹
3. 解压`.pak`档案或直接加载`.po`文件
4. 在编辑器中编辑字符串或使用**"用AI翻译"**
5. 导出完成的`.po`文件以重新导入游戏
